# Setup

## Platform 

* operating system : win10
* gvim : VI-Improved 8.1

## Usage 

### Windows
* Open gvim and then type `:echo $MYVIMRC` in normal mode,`vimrc's path` will show in the status line
* Use the `_vimrc` in this repo to overwrite the `defautl _vimrc`
* enjoy the gvim
